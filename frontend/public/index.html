# Start Building

Connect to PumpChain in 3 steps

The fastest SVM-powered chain for DeFi and Gaming. Get started in under 2 minutes.

1

Get Backpack Wallet

Download and install the Backpack wallet extension for your browser or mobile device.

Get Backpack ↗
2

Set Custom RPC

Open Backpack Settings → Solana → RPC Connection → Custom, then paste the PumpChain RPC URL.

3

You're Connected

Save and you're all set. Use the Launchpad to deploy tokens, trade on the DEX, or explore the chain.
